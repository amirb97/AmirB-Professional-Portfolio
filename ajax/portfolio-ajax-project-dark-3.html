Take A Hike

Project Description:

Website that allows user to find local hiking trails and weather forecast based off user input. Uses Google Maps APIs, OpenWeather API, and Tailwind for styling

Project Details:

  • GitHub Repo:https://github.com/amirb97/take-a-hike
  • Live Link:https://amirb97.github.io/take-a-hike/
  • Technologies:HTML, JavaScript, Tailwind, Google(Maps API, Places Library, Geocoder API, OpenWeather API)
